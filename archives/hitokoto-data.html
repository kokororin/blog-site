一言同步数据

·
日记 hitokoto 2015-07-15

简单来说，一言（ヒトコト）指的是就是一句话，可以是动漫中的台词，可以是小说中的语句，也可以是网络上的各种小段子。
或是感动，或是开心，又或是单纯的回忆，来到这里，留下你所喜欢的那一句句话，与大家分享，这就是一言存在的目的。

但是官方服务器比较慢（** 同时有倒闭的威胁 **），于是把它同步到本地。提供几个 API：

HTML TABLE
JSONP

参数说明
id 一言的 ID，可以通过 http://hitokoto.us/view/{ID}.html 查看指定条目
hitokoto 一言的正文部分，不包含前后引号
cat 分类，对应关系如下
a Anime - 动画
b Comic - 漫画
c Game - 游戏
d Novel - 小说
e 原创
f 来自网络
g 其他
catname 分类 cat 对应的名字
author 一言的投稿人
source 一言的出处，可能为空
date 一言的投稿日期
