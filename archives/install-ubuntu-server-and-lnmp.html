安装Ubuntu Server并配置LNMP环境

·
科技 linux php nginx virtualbox ubuntu 2015-08-18

这篇文章给大家介绍一下如何在Windows系统下的Virtual Box虚拟机软件中安装Ubuntu Server系统，并配置一个Linux+Nginx+Mysql+PHP，本文仅适合新手。

准备工作

  • 下载Virtualbox并安装
  • 在网易开源镜像站下载Ubuntu Server 14.04

安装系统

  • 由于VBox默认的虚拟电脑位置位于系统分区，建议移到其他分区。

  • 新建一个虚拟电脑内存选择512M，磁盘大小选择20GB，其他配置保持默认即可。

  • 启动刚刚创建的虚拟电脑，选择下载的iso文件。
  • 选择语言English

  • 回车选择Install

  • 下一步照样选择English

  • 地区推荐选择HongKong

  • 下一步将会检测键盘布局，选择No

  • 下一步回车
  • 再次回车，之后等待安装程序配置，可能需要一段时间

  • 配置Hostname，默认Ubuntu就可以
  • 配置用户名，我这里使用kotori，下一步回车
  • 配置密码，输入kotori，询问密码弱的话不用理会，选择Yes
  • 是否加密home，选择No

  • 询问时区是否正确，选择Yes
  • 下面将开始分区，选择Manual，之后选择刚刚创建的虚拟磁盘


  • 是否创建分区表，选择Yes
  • 选择Create a new partition，输入21.0GB，分区类型选择Primary，位置选择Begining，然后选择Done setting up the partition

  • 接下来还剩余一个475MB的空间，把该空间创建为swap，与上一步大致一样，类型选择Logical，位置选择End，Use as 选择swap area

  • 是否写入选择Yes，接下来等待系统安装，可能会比较慢，请耐心等待
  • 安装完后，询问是否使用HTTP代理，直接回车
  • 询问是否自动更新，选择No automatic updates
  • 使用空格键选择OpenSSH Server，回车

  • 最后一步啦，选择Yes，稍等一会会自动重启，到这里Ubuntu Server正式安装完成
