Chrome 扩展程序收集

·
科技 chrome 2015-09-07

必备类

Undo Close Tab：可以恢复最近关闭的标签页

Neat Bookmarks：把书签放到扩展区域，方便打开

SimpleExtManager：一键管理所有扩展

Speed Dial：优化 chrome 的新建标签页，可以放一堆自己常去的站点

Proxy SwitchySharp：前身是 Proxy Switchy! ，功能到界面都是一样的，可以快速切换代理，并且可以根据 pac 脚本实现自动翻墙

RSS Subscription Extension：在地址栏上添加一键订阅的 rss 按钮

Search by Image (by Google)：在图片右下角显示一个 icon，点击一键去 google 图搜图

浏览加强

IE tab：可以切换到 IE 内核，对付那些不支持 chrome 的网银

uBlock Origin：广告屏蔽

搜索直达：搜索结果再也无需跳转，直接打开目标网站！

下载类

Thunder,QQDownload,FlashGet Files：, 提供迅雷下载支持，地址解密。自动把页面里的迅雷，快车 Flashget, 旋风链接替换为真实地址，使用你自己喜欢的下载方式来下载，让 chrome 支持专用链下载，是一款特实用的下载助手

Chrono 下载管理器：Chrono 下载管理器是 Chrome 浏览器下第一款 (也是唯一一款) 功能全面的下载管理工具。Chrono 接管你在 Chrome 中的所有下载，你的所有下载管理工作都在浏览器中完成，而不需要安装另外的程序。Chrono 与 Chrome 浏览器紧密地整合在一起，添加了对浏览器菜单、工具栏和快捷键的支持

码农必备

Firebug：不如 Firefox 版

EyeDroper：浏览器窗口内取色，看到喜欢的配色可以查一下颜色代码留存以后用

MeasureIt：浏览器窗口内测距，用于测量元素的大小

Window Resizer：可以快速调整浏览器窗口的尺寸，用于观察网站页面宽度

Awesome Screenshot：个人感觉最好用的截图工具

IP Whois & Flags Chrome & Websites Rating：在地址栏显示网站所在地区的国旗，还可以显示 RP 值和 alexa 排名

PageMonitor：页面监控，如果内容有更新就通知你，我就是用这个插件在苹果官方商店上抢到了 iphone4，可用的地方很多，比容用来监控竞争对手的改版……

wappalyzer：Chrome 探测器，可以探测正在使用的开源软件或者 js 类库，开发者必备

冷门类

slim：思灵文本编辑器 (Slim Text) 是一款运行在浏览器中的轻量级文本编辑器

Start Today：你打开的不是空白页，是独一无二的今天

Web Timer：记录你在每个网站所花时间

SimilarWeb：寻找相似网站的扩展程序，例如访问 youtube，想浏览其他类似 youtube 的网站，点下图标，即可得到类似网站列表，准确度还是很高的，推荐使用

为什么你们就是不能加个空格呢？：一个台湾人写的一个很有意思的扩展，强迫症和处女座的福音

Secure Shell：懂得人都知道是干嘛的
