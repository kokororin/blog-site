Archlinux 安装 gnome 图形界面的方法

·
科技linuxarchlinux2014-10-21

基本系统的安装方法就不说了，参照 wiki（https://wiki.archlinux.org/index.php/Beginners'_guide）即可。

下面说说如何安装 gnome。

1、装 X

pacman -S xorg-server xorg-server-utils xorg-xinit xorg-utils

2、安装显卡驱动（intel）

pacman -S xf86-video-intel

3、安装触摸板和声卡

pacman -S xf86-input-synaptics alsa-utils

4、安装 gnome

pacman -S gnome gdm

5、安装 gnome-extra（可选）

pacman -S gnome-extra

6、设置服务

systemctl enable gdm
systemctl start gdm.service
