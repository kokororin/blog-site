adb 删除乱码文件 (夹)

·
科技 Android adb 2018-04-18

emmmm… 万年不水文

事情是这样的，在 Windows 下拷贝了一个中文目录进去，结果在手机上看就乱码了，而且目录里的文件都看不到，于是搜索找到一个 Linux 下删除乱码的办法：

adb shell
ls -i

看到类似的输出：

 6283265 鏂板缓\346\226

然后使用命令删除：

find -inum 6283265 -exec rm -rf {} \;
